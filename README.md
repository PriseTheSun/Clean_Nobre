# Lavanderia Clean Nobre

Sistema de agendamento online para serviços de lavanderia com coleta e entrega.

## Funcionalidades

- Agendamento de coleta via interface web
- Servicios: lavagem a seco, lavagem premium, passadoria, remoção de manchas
- Planos: Essencial, Premium, Família
- WhatsApp para contato direto
- Consentimento de cookies

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Vite
- Framer Motion (animações)
- Lucide React (ícones)

## Getting Started

```bash
npm install
npm run dev
```
